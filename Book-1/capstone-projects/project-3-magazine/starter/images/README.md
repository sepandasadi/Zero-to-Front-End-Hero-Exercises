# Images for Magazine Article Layout

## Required Images

### 1. Hero Image
- **File**: `hero-image.jpg`
- **Size**: 1200x600px (2:1 aspect ratio recommended)
- **Purpose**: Main header image for the article
- **Sources**: [Unsplash](https://unsplash.com), [Pexels](https://pexels.com)

### 2. Author Photo
- **File**: `author.jpg`
- **Size**: 200x200px (will be displayed as circle)
- **Purpose**: Author headshot
- **Sources**: [UI Faces](https://uifaces.co), or use a placeholder

### 3. Article Figures (3+ images)
- **Files**: `figure-1.jpg`, `figure-2.jpg`, `figure-3.jpg`, etc.
- **Size**: 800x600px (or appropriate for content)
- **Purpose**: In-article images with captions
- **Should relate to article content**

### 4. Image Grid/Gallery (2-4 images)
- **Files**: `gallery-1.jpg` through `gallery-4.jpg`
- **Size**: 600x400px (consistent aspect ratio)
- **Purpose**: Image gallery section

## Using Placeholders

During development, use placeholder images:

```html
<img src="https://via.placeholder.com/1200x600" alt="Hero image">
<img src="https://via.placeholder.com/200x200" alt="Author photo">
<img src="https://via.placeholder.com/800x600" alt="Figure image">
```

## Image Optimization

Before final submission:
1. Resize images to appropriate dimensions
2. Compress with [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app)
3. Convert to WebP for better performance (optional)
4. Aim for < 200KB per image

If using free stock photos, check licenses and provide attribution if required.
