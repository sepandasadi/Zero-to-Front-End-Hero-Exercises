# Project 3: Magazine Article Layout - Evaluation Rubric

**Total Points: 100**

---

## Typography & Readability (25 points)

### Font System (10 points)
- **Excellent (9-10):** Professional font pairing, proper type scale using CSS variables, excellent readability
- **Good (7-8):** Good font choices, mostly consistent sizing
- **Needs Improvement (5-6):** Limited typography system, inconsistent sizing
- **Unsatisfactory (0-4):** Poor font choices, no system

### Line Length & Spacing (8 points)
- **Excellent (7-8):** Optimal line length (60-75ch), perfect paragraph spacing, excellent rhythm
- **Good (5-6):** Acceptable line length, decent spacing
- **Needs Improvement (3-4):** Line length too long/short, spacing inconsistent
- **Unsatisfactory (0-2):** Unreadable line lengths, no spacing consideration

### Drop Cap & Special Elements (7 points)
- **Excellent (6-7):** Beautiful drop cap, pull quotes styled distinctively, proper blockquotes
- **Good (4-5):** Drop cap present, quotes styled
- **Needs Improvement (2-3):** Basic implementation
- **Unsatisfactory (0-1):** Missing or broken

---

## Layout & Grid (25 points)

### CSS Grid Implementation (12 points)
- **Excellent (11-12):** Advanced grid layout, sidebar implemented perfectly, responsive grid areas
- **Good (9-10):** Solid grid usage, minor issues
- **Needs Improvement (6-8):** Basic grid, significant layout problems
- **Unsatisfactory (0-5):** No grid or severely broken

### Multi-Column Text (5 points)
- **Excellent (5):** Perfect multi-column sections, no awkward breaks
- **Good (4):** Multi-column works with minor issues
- **Needs Improvement (2-3):** Multi-column present but breaks awkwardly
- **Unsatisfactory (0-1):** Missing or broken

### Sidebar & Related Content (8 points)
- **Excellent (7-8):** Well-designed sidebar, properly positioned, quality content
- **Good (5-6):** Sidebar present and functional
- **Needs Improvement (3-4):** Basic sidebar, positioning issues
- **Unsatisfactory (0-2):** Missing or broken

---

## Visual Elements (20 points)

### Figures & Images (8 points)
- **Excellent (7-8):** 3+ high-quality figures, beautiful captions, proper semantic HTML, responsive images
- **Good (5-6):** Figures present with captions
- **Needs Improvement (3-4):** Basic images, missing captions
- **Unsatisfactory (0-2):** Poor image handling

### Pull Quotes & Blockquotes (6 points)
- **Excellent (5-6):** Distinctive pull quotes, proper blockquote styling, enhances content
- **Good (4):** Pull quotes present and styled
- **Needs Improvement (2-3):** Basic implementation
- **Unsatisfactory (0-1):** Missing or indistinguishable

### Image Grid/Gallery (6 points)
- **Excellent (5-6):** Beautiful grid layout, equal heights, responsive
- **Good (4):** Grid present and functional
- **Needs Improvement (2-3):** Basic grid with issues
- **Unsatisfactory (0-1):** Missing or broken

## Code Quality & Accessibility (5 points)

### HTML Quality (2 points)
- **Excellent (2):** Semantic HTML, valid, proper structure
- **Unsatisfactory (0-1):** Poor HTML quality

### CSS Organization (2 points)
- **Excellent (2):** CSS variables used properly, well-organized, commented

## Bonus Points (Up to +15)

- **Advanced Features** (+3-5): Footnotes, image zoom, related articles
- **Exceptional Design** (+3-5): Professional-quality visual design
- **Performance** (+2-3): Optimized images, fast loading, excellent Lighthouse scores

---

## Grading Scale

- **90-100 points:** Excellent - Portfolio quality, professional magazine layout
- **80-89 points:** Good - Solid project, meets all requirements
- **70-79 points:** Satisfactory - Functional but needs polish
- **60-69 points:** Needs Improvement - Missing features or significant issues
- **Below 60:** Unsatisfactory - Does not meet minimum requirements
