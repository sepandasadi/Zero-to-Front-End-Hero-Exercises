# Task Manager API - Challenge Project

Complete the full-stack task manager API following the 8 phases outlined in the challenge README.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
MONGODB_URI=mongodb://localhost:27017/task-manager
JWT_SECRET=your-secret-key-change-in-production
PORT=3000
NODE_ENV=development
```

3. Start development server:
```bash
npm run dev
```
