### Required Structure

- [ ] Main heading (`<h1>`) with your name

- [ ] Subheading (`<h2>`): "About Me"
- [ ] 2-3 paragraphs introducing yourself

- [ ] Subheading (`<h2>`): "My Interests" (or "What I Love")

- [ ] Subheading (`<h2>`): "Skills I'm Learning" or "My Goals"
- [ ] Ordered list (`<ol>`) of skills or goals

#### Contact/Connect Section:
- [ ] Subheading (`<h2>`): "Get in Touch" or "Let's Connect"
- [ ] Paragraph with links to:
  - Email (mailto link)
  - GitHub (opens in new tab)
  - LinkedIn (opens in new tab)
  - Or other social profiles

#### Visual Elements:
- [ ] At least one image (photo, avatar, or placeholder)
- [ ] All images have appropriate `alt` text

**Code Quality:**
- [ ] All tags properly nested
- [ ] Proper indentation (2 or 4 spaces consistent)
- [ ] Helpful comments for each major section
- [ ] No broken tags or syntax errors

- [ ] Descriptive alt text for all images
