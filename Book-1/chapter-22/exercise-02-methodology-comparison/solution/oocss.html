OOCSS Structure Notes:

  • Structure Classes (Layout & Spacing):
    • .box - Generic container
    • .box-padding - Padding structure
    • .flex - Flexbox layout
    • .mb-2, .mb-3 - Margin bottom spacing
  • Skin Classes (Appearance):
    • .box-rounded - Rounded corners
    • .box-shadow - Shadow effect
    • .btn-primary - Primary button color
    • .text-price--sale - Sale price color
  • Media Object:
    • .media__image - Reusable image container
    • .media__image--cover - Object-fit cover modifier

Pros: Highly reusable classes, separation of concerns, flexible combinations

Cons: More classes in HTML, requires understanding of the system
