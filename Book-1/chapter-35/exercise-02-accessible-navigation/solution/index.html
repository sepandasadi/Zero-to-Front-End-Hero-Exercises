Welcome to Accessible Navigation

This navigation is fully accessible and works with keyboard, mouse, and screen readers.

Accessibility Features:

  • ✓ Navigate with Tab key
  • ✓ Press Enter to activate links
  • ✓ Press Escape to close mobile menu
  • ✓ Screen reader announces menu state
  • ✓ Focus is trapped in mobile menu
  • ✓ Visible focus indicators
  • ✓ Skip to main content link

Test it:

  1. Unplug your mouse and navigate with keyboard only
  2. Use a screen reader (NVDA, VoiceOver, JAWS)
  3. Check with WAVE or axe DevTools
  4. Zoom to 200% and verify layout
