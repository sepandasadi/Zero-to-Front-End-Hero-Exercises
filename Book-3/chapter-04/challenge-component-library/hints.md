# Challenge: Component Library - Hints

## Project Structure

```
component-library/
├── scss/
│   ├── abstracts/
│   │   ├── _variables.scss    (design tokens)
│   │   ├── _functions.scss    (helpers)
│   │   └── _mixins.scss       (reusable patterns)
│   ├── base/
│   │   ├── _reset.scss
│   │   └── _typography.scss
│   ├── components/
│   │   ├── _button.scss
│   │   ├── _card.scss
│   │   ├── _form.scss
│   │   ├── _nav.scss
│   │   └── _modal.scss
│   ├── utilities/
│   │   └── _generated.scss    (auto-generated utilities)
│   ├── themes/
│   │   ├── _light.scss
│   │   └── _dark.scss
│   └── main.scss
├── demo/
│   └── index.html
└── README.md
```

- [ ] Generate utility classes
- [ ] Implement light theme
- [ ] Implement dark theme

### Phase 4: Documentation (30 min)
- [ ] README with usage examples
