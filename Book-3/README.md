### **Prerequisites:**

**Required:**
- Complete Volumes 1 & 2 (HTML, CSS, JavaScript)
- Node.js (v18+) and npm
- Git and GitHub account
- VS Code or preferred editor
- Modern browser (Chrome recommended)

**Recommended:**
- Basic terminal/command line skills
- Familiarity with package managers (npm/yarn)
- GitHub account for deployment
- Vercel/Netlify account (free tier)

### **Setup:**

```bash
# Clone the repository
git clone https://github.com/yourusername/Zero-to-Front-End-Hero-Exercises.git

# Navigate to Book 3
cd Zero-to-Front-End-Hero-Exercises/Book-3

npm install -g npm@latest

# For React exercises (Vite)

# For Vue exercises (Vite)
npm create vite@latest my-vue-app -- --template vue

# For Tailwind setup
```
