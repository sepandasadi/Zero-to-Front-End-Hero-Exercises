## ðŸ”§ Optimization Checklist

### Before Adding to Solution:

1. **Resize to exact dimensions**
   - Instructor: 600x600px
   - Students: 200x200px each

2. **Compress aggressively**
   - Use TinyPNG or Squoosh
   - Target file sizes listed above
   - Aim for 70-80% quality

3. **Convert to WebP (optional)**
   - Better compression than JPG
   - Provide JPG fallback in HTML

4. **Test loading**
   - Ensure fast load times
   - Use lazy loading attribute

5. **Alt text**
   - Write descriptive alt text
   - Include person's name and role
