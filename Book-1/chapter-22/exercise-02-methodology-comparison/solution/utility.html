Utility-First Structure Notes:

  • Layout Utilities:
    • .flex, .block - Display types
    • .items-center - Alignment
    • .w-full - Full width
    • .overflow-hidden - Overflow control
  • Spacing Utilities:
    • .p-6 - Padding (1.5rem)
    • .mb-2, .mb-4 - Margin bottom
    • .ml-2, .ml-3 - Margin left
  • Color Utilities:
    • .bg-white, .bg-blue-600 - Backgrounds
    • .text-gray-900, .text-red-600 - Text colors
  • Typography Utilities:
    • .text-xl, .text-base - Font sizes
    • .font-bold, .font-semibold - Font weights
  • Effects Utilities:
    • .rounded-lg, .rounded-md - Border radius
    • .shadow-md - Box shadow
    • .hover:bg-blue-700 - Hover states

Pros: Very fast development, consistent spacing/colors, minimal CSS to write

Cons: HTML becomes verbose, many classes per element, less semantic
