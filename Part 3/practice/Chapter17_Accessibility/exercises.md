# Chapter 17 — Accessibility (Applied) Exercises

1) Fix `aria-examples.html` to add proper landmarks, headings, and interactive controls.
2) In `keyboard-trap.html`, implement a focus trap and restore focus to the opener on close.
3) Apply visible focus styles from `focus-visible.css` and validate keyboard flows.
4) Use `contrast-check.js` to evaluate color pairs you use and pick a set with ≥ 4.5:1 contrast.
5) (Optional) Add `axe-playwright.spec.ts` to CI and fail the build on violations.
