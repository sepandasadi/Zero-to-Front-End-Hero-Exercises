IDs and Classes

This paragraph has a unique ID of "introduction". IDs must be unique on the page and can be used for navigation links, styling with CSS, or selecting with JavaScript.

This paragraph has a class of "highlight". Multiple elements can share the same class, making classes perfect for applying consistent styling or behavior to groups of elements.

This is another paragraph with the "highlight" class. Notice how multiple elements can have the same class, unlike IDs which must be unique!

This paragraph has TWO classes: "highlight" and "important". You can apply multiple classes to a single element by separating them with spaces. This is super useful when you want to combine different styles!
