# Accessibility Statement for TechStore

**Last updated:** [Date]

## Our Commitment

TechStore is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

The [Web Content Accessibility Guidelines (WCAG)](https://www.w3.org/WAI/WCAG21/quickref/) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

**TechStore is fully conformant with WCAG 2.1 level AA.**

"Fully conformant" means that the content fully conforms to the accessibility standard without any exceptions.

## Technical Specifications

Accessibility of TechStore relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

- HTML5
- WAI-ARIA
- CSS3
- JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

## Accessibility Features

TechStore includes the following accessibility features:

### For Screen Reader Users
- Semantic HTML landmarks for easy navigation
- ARIA labels and descriptions
- Skip links to bypass repetitive content
- Proper heading hierarchy
- Alternative text for images

### For Keyboard Users
- Full keyboard navigation support
- Visible focus indicators
- No keyboard traps
- Logical tab order

### For Users with Low Vision
- High contrast text (4.5:1 for normal text, 3:1 for large text)
- Resizable text up to 200% without loss of functionality
- No reliance on color alone to convey information

### For Users with Cognitive Disabilities
- Clear and consistent navigation
- Descriptive headings and labels
- Error messages with suggestions
- Sufficient time to complete tasks
