## üìä Performance Targets

**Good Performance:**
- Lighthouse score: 90+
- LCP: < 2.5s
- INP: < 200ms
- CLS: < 0.1
- Total bundle: < 200KB (gzipped)
- Images: WebP/AVIF format

**Excellent Performance:**
- Lighthouse score: 95+
- LCP: < 1.5s
- INP: < 100ms
- CLS: < 0.05
- Total bundle: < 100KB
- All assets on CDN
