# Product Landing Page - Starter Files

## What's Included

- `index.html` - Basic HTML structure with TODOs
- `css/styles.css` - CSS reset and starter styles
- `images/` - Folder for your images (see images/README.md)

## Getting Started

1. **Choose Your Product**
   - Pick a product or service to feature (real or fictional)
   - Examples: SaaS tool, mobile app, physical product, service

2. **Gather Assets**
   - Find or create images (see `images/README.md` for guidance)
   - Choose a color scheme (update CSS variables)
   - Select fonts (Google Fonts recommended)

3. **Build in Order**
   - Start with HTML structure (semantic elements!)
   - Add CSS styling section by section
   - Make it responsive last

4. **Test Continuously**
   - Test in browser after each section
   - Check responsive behavior early
   - Validate HTML and CSS regularly

## Recommended Workflow

### Day 1: HTML Structure (1-2 hours)
- [ ] Complete HTML for all 7 sections
- [ ] Use semantic elements
- [ ] Add placeholder content
- [ ] Validate HTML

### Day 2: Desktop Styling (2-3 hours)
- [ ] Style hero section
- [ ] Style features with Grid
- [ ] Create pricing cards
- [ ] Add testimonials
- [ ] Style FAQ section

### Day 3: Responsive & Polish (1-2 hours)
- [ ] Add media queries
- [ ] Test on mobile and tablet
- [ ] Add animations and transitions
- [ ] Optimize images
- [ ] Final testing

## Tips

- **Start simple:** Get basic layout working before adding fancy features
- **Use browser DevTools:** Inspect and experiment with styles
- **Save often:** Commit your changes frequently
- **Ask for help:** Check `hints.md` when stuck
