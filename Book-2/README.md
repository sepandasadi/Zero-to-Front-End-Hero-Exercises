### Common JavaScript Issues & Solutions

**Problem: "Uncaught ReferenceError: X is not defined"**
- âœ… Check spelling of variable/function names
- âœ… Ensure variable is declared before use
- âœ… Check scope - variable may not be accessible where you're using it
- âœ… For DOM elements, ensure script runs after HTML loads (put `<script>` before `</body>`)

**Problem: "Uncaught TypeError: Cannot read property 'X' of undefined"**
- âœ… The object/variable is `undefined` or `null`
- âœ… Check that API calls or queries succeeded
- âœ… Use optional chaining: `object?.property` (Chapter 7)
- âœ… Add null checks: `if (object) { ... }`

**Problem: "My async code doesn't work / returns undefined"**
- âœ… Make sure you're using `await` with async functions
- âœ… Check that the function is declared with `async` keyword
- âœ… Use `.then()` or `await` to handle promises
- âœ… Add `.catch()` or try/catch to see errors

**Problem: "npm install fails"**
- âœ… Check Node.js version: `node --version` (need 18+)
- âœ… Delete `node_modules/` and `package-lock.json`, then retry
- âœ… Clear npm cache: `npm cache clean --force`
- âœ… Check for typos in package.json
- âœ… Try with elevated privileges if permission errors

**Problem: "My code works in the console but not in the file"**
- âœ… Check file is saved
- âœ… Ensure script tag is correctly linked in HTML
- âœ… Check for typos in file path
- âœ… Look for syntax errors that prevent script from running
- âœ… Check browser console for error messages

**Problem: "Event listener isn't firing"**
- âœ… Ensure element exists when addEventListener is called
- âœ… Check event name spelling (`'click'`, not `'onclick'`)
- âœ… Verify selector targets the correct element
- âœ… Use event delegation for dynamically added elements

**Problem: "Tests are failing"**
- âœ… Read the test error message - it shows what's expected vs actual
- âœ… Check that your function returns (not just logs) the value
- âœ… Ensure you're handling edge cases (empty arrays, null values)
- âœ… Run `npm test` to see detailed error output

### Debugging Workflow

**Step-by-Step Debugging:**
1. **Identify where the problem occurs** - Use `console.log()` to track execution
2. **Check values at each step** - Log variables to see what they contain
3. **Use browser DevTools** - Set breakpoints and step through code
4. **Isolate the problem** - Comment out code to narrow down the issue
5. **Test small pieces** - Verify each function works independently
6. **Compare with solution** - See what's different in the working version
