## üéì Bonus Challenge

Set up terminal aliases (shortcuts for longer commands):

**macOS/Linux** (add to `~/.bashrc` or `~/.zshrc`):
```bash
alias ll='ls -la'
alias ..='cd ..'
alias projects='cd ~/Desktop/projects'
```

**Windows** (PowerShell profile):
```powershell
function projects { cd C:\Users\YourName\Desktop\projects }
```
