# Project 2: Weather Dashboard

**Difficulty:** ⭐⭐⭐ API Integration  
**Estimated Time:** 6-8 hours  
**Skills Practiced:** Async/Await, Fetch API, Error Handling, ES Modules, API Integration

## Project Overview

Create a multi-city weather dashboard that fetches real-time data from the OpenWeather API. This project builds on Project 1's patterns while introducing asynchronous JavaScript, API integration, and advanced error handling.

## Core Features
- Search and add cities
- Display current weather (temp, conditions, humidity, wind)
- Show 5-day forecast
- Save favorite cities (localStorage)
- Auto-refresh every 10 minutes
- Temperature unit toggle (°C/°F)
- Loading states and error messages
- Responsive grid layout

## Milestones
1. **API Setup & Basic Fetch** (2 hours)
2. **Multi-City Dashboard** (2 hours)
3. **5-Day Forecast** (1.5 hours)
4. **Polish & Features** (2.5 hours)
