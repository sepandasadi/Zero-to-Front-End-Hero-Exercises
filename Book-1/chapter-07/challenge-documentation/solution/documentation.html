Introduction

Last updated: January 15, 2024

Welcome to the complete guide to HTML and CSS fundamentals! Whether you're just starting your web development journey or refreshing your knowledge, this documentation will provide you with a solid foundation.

Overview

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the two fundamental technologies for building web pages:

  • HTML provides the structure and content
  • CSS handles the visual presentation and layout

Together, they form the backbone of every website you visit.

Why They Matter

Understanding HTML and CSS is essential because:

  1. They're the foundation of web development
  2. Every web developer must know them
  3. They're relatively easy to learn
  4. You can start building real projects immediately
  5. They provide a gateway to more advanced topics

What You'll Learn

This guide covers:

  • HTML document structure and syntax
  • Common HTML elements and their uses
  • Semantic HTML best practices
  • CSS selectors and specificity
  • Common CSS properties and values
  • How to combine HTML and CSS effectively

HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements represented by tags.

HTML tells browsers how to display contentâ€”it's not a programming language, but a markup language.

Document Structure

Every HTML document has a basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  <!-- Content goes here -->
</body>
</html>

Let's break this down:

  • <!DOCTYPE html> - Declares this is an HTML5 document
  • <html> - Root element of the page
  • <head> - Contains metadata and page information
  • <body> - Contains the visible page content

Common Elements

HTML provides many elements for different types of content:

Headings

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Paragraphs and Text

<p>This is a paragraph.</p>
<strong>Important text</strong>
<em>Emphasized text</em>

Links and Images

<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Description">

Lists

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Best Practices

  • Always include alt text for images
  • Use proper heading hierarchy (don't skip levels)
  • Keep your code properly indented

CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML documents. It controls colors, fonts, layouts, spacing, and all visual aspects of your web page.

CSS separates presentation from structure, making your code cleaner and easier to maintain.

Selectors

CSS selectors target HTML elements you want to style:

Element Selector

p {
  color: blue;
  font-size: 16px;
}

Class Selector

.highlight {
  background-color: yellow;
  font-weight: bold;
}

ID Selector

#header {
  background-color: navy;
  color: white;
}

Common Properties

CSS offers hundreds of properties. Here are the most essential ones:

Colors and Backgrounds

element {
  color: #333;
  background-color: #f0f0f0;
}

Typography

element {
  font-family: Arial, sans-serif;
  font-size: 18px;
  font-weight: bold;
  line-height: 1.6;
}

Spacing

element {
  margin: 20px;
  padding: 15px;
}

Complete Example

Here's a complete CSS stylesheet example:

body {
  font-family: 'Segoe UI', sans-serif;
  line-height: 1.6;
  color: #333;
  margin: 0;
  padding: 20px;
}

h1, h2, h3 {
  color: #2c3e50;
}

a {
  color: #3498db;
  text-decoration: none;
}

a:hover {
  text-decoration: underline;
}

Putting it Together

Complete Example

Let's combine HTML and CSS to create a simple but complete web page:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Webpage</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      max-width: 800px;
      margin: 40px auto;
      padding: 20px;
      line-height: 1.6;
    }

    header {
      background-color: #2c3e50;
      color: white;
      padding: 20px;
      border-radius: 5px;
    }

    h1 { margin: 0; }

    nav { margin: 20px 0; }

    nav a {
      margin-right: 15px;
      color: #3498db;
      text-decoration: none;
    }

    article {
      background-color: #ecf0f1;
      padding: 20px;
      border-radius: 5px;
      margin: 20px 0;
    }
  </style>
</head>
<body>
  <header>
    <h1>Welcome to My Website</h1>
  </header>

  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
  </nav>

  <main>
    <article>
      <h2>About This Site</h2>
      <p>This is a simple example of HTML and CSS working together.</p>
    </article>
  </main>
</body>
</html>

Tips and Tricks

  • Start with HTML structure first, then add CSS
  • Use external stylesheets for larger projects
  • Organize your CSS logically (layout, typography, components)
  • Comment your code to explain complex sections
  • Use browser DevTools to experiment
  • Test in multiple browsers

Glossary

Element
An HTML component defined by tags (e.g., <p>)
Attribute
Additional information about an element (e.g., href)
Selector
A pattern used to target elements for styling
Property
A CSS characteristic you can style (e.g., color)
Value
The setting for a CSS property (e.g., red)
