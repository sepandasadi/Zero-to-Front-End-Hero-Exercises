BEM Structure Explanation

Block: .profile-card

The main standalone component. It represents the entire user profile card.

Elements: .profile-card__*

  • .profile-card__header - Top section with avatar and info
  • .profile-card__avatar - User's profile image
  • .profile-card__info - Container for name and title
  • .profile-card__name - User's name
  • .profile-card__title - User's job title
  • .profile-card__bio - Biography section
  • .profile-card__bio-text - The bio paragraph text
  • .profile-card__stats - Statistics container
  • .profile-card__stat - Individual stat item
  • .profile-card__stat-number - Stat value
  • .profile-card__stat-label - Stat label
  • .profile-card__actions - Button container
  • .profile-card__badge - Featured badge

Modifier: .profile-card--featured

A variation of the profile card that has special styling (gold border, badge).

Separate Block: .button

Buttons are separate blocks because they're reusable components that can exist independently.

  • .button--primary - Primary button style
  • .button--secondary - Secondary button style
