Real-World CSS Architecture

ITCSS + BEM + OOCSS Working Together

What to Build

Create a production-ready component library with:

  • ITCSS Structure: 7 layers (settings, tools, generic, elements, objects, components, utilities)
  • OOCSS Objects: Layout patterns (o-container, o-media, o-grid)
  • BEM Components: UI components (c-button, c-card, c-form, c-navigation)

Architecture Layers

scss/
├── 1-settings/      # Variables (colors, typography, spacing)
├── 2-tools/         # Mixins & functions
├── 3-generic/       # Reset
├── 4-elements/      # HTML elements
├── 5-objects/       # OOCSS layout patterns (o-*)
├── 6-components/    # BEM components (c-*)
├── 7-utilities/     # Helper classes (u-*)
└── main.scss        # Import orchestration
