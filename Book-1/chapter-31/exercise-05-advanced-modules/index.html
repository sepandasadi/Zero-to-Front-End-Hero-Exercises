üéØ Module Patterns Used

Named Exports

Export specific functions/variables

export { add, PI } import { add } from './math.js'

Default Export

Single main export per module

export default Logger import Logger from './logger.js'

Namespace Import

Import everything as object

import * as math from './math.js' math.add(1, 2)

Barrel Pattern

Re-export from single entry

// index.js export * from './math.js'

Dynamic Import

Load modules on demand

const mod = await import('./data.js') mod.doSomething()

Mixed Exports

Combine default + named

import config, { API_URL } from './config.js'
