# Images Folder

Place your project images in this folder.

## Required Images

### Hero Section
- **hero-background.jpg** (1920x1080px or larger)
  - Background image for hero section
  - Should be high quality but optimized
  - Free sources: [Unsplash](https://unsplash.com), [Pexels](https://pexels.com)

### Product/Features
- **product-screenshot.png** (800x600px recommended)
  - Main product image or mockup
- **feature-icon-1.svg** through **feature-icon-6.svg**
  - Icons for each feature
  - Can use Font Awesome instead
  - Free icons: [Heroicons](https://heroicons.com), [Feather Icons](https://feathericons.com)

### Testimonials
- **testimonial-1.jpg** through **testimonial-3.jpg** (or more)
  - Circular crop recommended (400x400px)
  - Professional headshots
  - Free sources: [UI Faces](https://uifaces.co), [Random User](https://randomuser.me)

## Image Optimization Tips

### Before Adding Images:
1. **Resize** images to appropriate dimensions (don't use 5000px images!)
2. **Compress** using tools like:
   - [TinyPNG](https://tinypng.com)
   - [Squoosh](https://squoosh.app)
   - [ImageOptim](https://imageoptim.com) (Mac)
3. **Convert** to WebP for better compression (optional)
4. **Name** files descriptively (hero-background.jpg, not IMG_1234.jpg)

### Recommended Sizes:
- **Hero backgrounds**: 1920x1080px (Full HD)
- **Feature icons**: 100x100px or use SVG
- **Product screenshots**: 800x600px
- **Testimonial photos**: 200x200px (displayed smaller)

## Using Placeholder Images

During development, you can use placeholder services:

```html
<!-- Example: Placeholder image -->
<img src="https://via.placeholder.com/1920x1080" alt="Hero background">
```

**Replace with real images before final submission!**
