## Question 1
**What is the difference between the Internet and the World Wide Web?**

a) They are the same thing
b) The Internet is the infrastructure; the Web is the information system built on it
c) The Web came before the Internet
d) The Internet is only for email

<details>
<summary>Answer</summary>
**b) The Internet is the infrastructure; the Web is the information system built on it**

The Internet is the global network of computers, while the World Wide Web is the system of interlinked documents and resources accessed via the Internet using HTTP/HTTPS.
</details>

---

## Question 2
**In the client-server model, what is the role of the client?**

a) To store all website data permanently
b) To request resources from servers and display them to users
c) To host websites for other users
d) To manage databases

<details>
<summary>Answer</summary>
**b) To request resources from servers and display them to users**

The client (typically a web browser) requests resources from servers and renders them for the user. The server stores and provides the resources.
</details>

---

## Question 3
**Which three technologies form the core of front-end web development?**

a) Python, Ruby, PHP
b) HTML, CSS, JavaScript
c) SQL, MongoDB, Redis
d) React, Vue, Angular

<details>
<summary>Answer</summary>
**b) HTML, CSS, JavaScript**

HTML provides structure, CSS provides styling, and JavaScript provides interactivity. React, Vue, and Angular are frameworks built on top of these core technologies.
</details>

## Question 4
**What does HTML stand for, and what is its primary purpose?**

a) Hyper Text Markup Language - for styling websites
b) High Transfer Markup Language - for data transfer
c) Hyper Text Markup Language - for structuring content
d) Hyperlink and Text Management Language - for creating links

<details>
<summary>Answer</summary>
**c) Hyper Text Markup Language - for structuring content**

HTML (Hyper Text Markup Language) is used to structure and organize content on web pages using elements and tags.
</details>

## Question 5
**What is CSS primarily used for?**

a) Making websites interactive
b) Storing user data
c) Styling and layout of web pages
d) Server-side logic

<details>
<summary>Answer</summary>
**c) Styling and layout of web pages**

CSS (Cascading Style Sheets) is used to control the visual presentation, styling, and layout of web pages.
</details>

## Question 6
**What role does JavaScript play in web development?**

a) It defines the structure of web pages
b) It adds interactivity and dynamic behavior
c) It styles elements on the page
d) It manages server databases

<details>
<summary>Answer</summary>
**b) It adds interactivity and dynamic behavior**

JavaScript is a programming language that adds interactivity, handles user events, manipulates the DOM, and enables dynamic content updates.
</details>

## Question 9
**What does HTTP stand for, and what is it used for?**

a) Hyper Text Transfer Protocol - for transferring web resources
b) High Tech Transfer Process - for downloading files
c) Hyperlink Text Transfer Protocol - for creating links
d) Hyper Text Transmission Protocol - for email

<details>
<summary>Answer</summary>
**a) Hyper Text Transfer Protocol - for transferring web resources**

HTTP is the protocol used for transferring data (HTML, CSS, JS, images, etc.) between clients and servers on the web.
</details>

## Question 10
**What are browser Developer Tools used for?**

a) Building websites from scratch
b) Inspecting, debugging, and analyzing web pages
c) Hosting websites
d) Writing code automatically

<details>
<summary>Answer</summary>
**b) Inspecting, debugging, and analyzing web pages**

Developer Tools (DevTools) let you inspect HTML/CSS, debug JavaScript, monitor network requests, check performance, and test responsive designs.
</details>
