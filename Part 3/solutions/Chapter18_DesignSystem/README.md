# Minimal Design System Starter

This starter includes:
- Design tokens (CSS variables) with light/dark themes
- Accessible Button and Dialog primitives
- Storybook stories with a11y addon

## Quick Start
```
npm i -D storybook @storybook/react-vite @storybook/addon-a11y @storybook/addon-essentials react react-dom typescript
# then run storybook
npx storybook init
npm run storybook
```
