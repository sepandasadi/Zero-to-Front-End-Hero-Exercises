# Exercise 1: Navigation and Exploration

## üéØ Objective
Master basic terminal navigation commands and understand your file system structure.

## üìù Instructions

1. **Open your terminal/command prompt**

2. **Complete these tasks** and document the commands you used:

   a) Display your current location (pwd/cd)
   b) List all files in the current directory
   c) Navigate to your home directory
   d) Navigate to your Desktop
   e) Go back to the previous directory
   f) Go up one level (parent directory)

3. **Create a navigation log** (`navigation-log.txt`) documenting:
   - Each command you used
   - What the output was
   - Where you ended up

## üí° Commands Reference

**macOS/Linux:**
```bash
pwd                # Print working directory
ls                 # List files
ls -la            # List all files (including hidden)
cd ~              # Go to home directory
cd ..             # Go up one level
cd /path/to/dir   # Go to specific directory
```

**Windows:**
```cmd
cd                # Show current directory
dir               # List files
cd %USERPROFILE%  # Go to home directory
cd ..             # Go up one level
cd \path\to\dir   # Go to specific directory
```
